# 企业设备领用与位置追溯系统 - 使用说明文档

## 1. 系统概述

### 1.1 系统简介

企业设备领用与位置追溯系统是一套专为企业资产管理设计的综合性解决方案，旨在实现资产从采购、领用、维护到报废的全生命周期管理，提供实时位置追踪和高效盘点功能，帮助企业提高资产管理效率，减少资产流失。

### 1.2 系统功能亮点

- 二维码扫码快速登记与查询
- 实时位置追踪与状态更新
- 移动端适配，支持盘点枪使用
- 完善的审批流程与权限控制
- 丰富的报表与数据分析功能
- 资产全生命周期管理

## 2. 系统登录与主界面

### 2.1 登录方式

1. 打开浏览器，输入系统网址
2. 在登录页面输入用户名和密码
3. 选择用户角色（管理员/部门主管/员工）
4. 点击登录按钮进入系统

### 2.2 主界面介绍

系统主界面包含以下主要元素：

- **顶部导航栏**：包含用户信息、通知中心、系统设置等
- **左侧菜单栏**：包含各功能模块入口
- **主内容区**：显示当前模块的详细信息
- **数据统计仪表盘**：展示关键指标和统计图表

## 3. 功能模块使用指南

### 3.1 资产管理

#### 3.1.1 资产录入

1. 点击左侧菜单「资产管理」→「资产录入」
2. 填写资产信息：资产名称、类型、品牌、型号、序列号等
3. 点击「生成二维码」自动生成唯一二维码
4. 上传相关附件（如采购发票、产品说明书等）
5. 点击「保存」完成资产录入

#### 3.1.2 资产查询与编辑

1. 点击左侧菜单「资产管理」→「资产列表」
2. 使用搜索条件筛选资产：资产名称、类型、状态、部门等
3. 在资产列表中点击具体资产查看详情
4. 点击「编辑」按钮修改资产信息
5. 点击「导出」按钮导出资产列表

### 3.2 二维码管理

#### 3.2.1 二维码生成与打印

1. 点击左侧菜单「二维码管理」→「生成二维码」
2. 选择需要生成二维码的资产范围（单个/批量）
3. 设置二维码格式（大小、样式）
4. 点击「生成」按钮
5. 点击「批量打印」按钮连接打印机打印二维码标签

#### 3.2.2 二维码标签规范

- 二维码标签应包含：资产编码、资产名称、品牌型号
- 标签材质建议使用防水、防油的不干胶
- 标签尺寸建议为：3cm×5cm
- 标签应牢固粘贴在资产明显位置

### 3.3 领用与归还

#### 3.3.1 资产领用

**方式一：管理员分配**
1. 点击左侧菜单「领用管理」→「资产分配」
2. 搜索并选择要分配的资产
3. 选择领用人和使用部门
4. 填写领用原因
5. 点击「确认分配」完成操作

**方式二：扫码领用**
1. 点击左侧菜单「扫码操作」→「领用登记」
2. 使用设备扫描资产二维码
3. 系统自动识别资产信息
4. 确认领用人信息
5. 点击「确认领用」完成操作

#### 3.3.2 资产归还

**方式一：管理员操作**
1. 点击左侧菜单「领用管理」→「资产归还」
2. 搜索并选择要归还的资产
3. 确认归还位置
4. 检查资产状态
5. 点击「确认归还」完成操作

**方式二：扫码归还**
1. 点击左侧菜单「扫码操作」→「归还登记」
2. 使用设备扫描资产二维码
3. 系统自动识别资产信息
4. 选择归还位置
5. 点击「确认归还」完成操作

### 3.4 位置管理

#### 3.4.1 位置信息维护

1. 点击左侧菜单「位置管理」→「位置设置」
2. 点击「新增」按钮添加新位置
3. 设置位置编码、位置名称、所属部门
4. 可以设置父位置，建立位置层级关系
5. 点击「保存」完成位置添加

#### 3.4.2 资产位置查询

1. 点击左侧菜单「位置管理」→「位置查询」
2. 选择或搜索特定位置
3. 系统显示该位置下的所有资产
4. 可查看资产详细信息和状态

### 3.5 维修管理

#### 3.5.1 维修申请

1. 点击左侧菜单「维修管理」→「维修申请」
2. 点击「新增申请」按钮
3. 选择需要维修的资产
4. 填写故障描述和维修要求
5. 指定维修负责人
6. 点击「提交申请」

#### 3.5.2 维修记录管理

1. 点击左侧菜单「维修管理」→「维修记录」
2. 查看所有维修申请及其状态
3. 更新维修进度和结果
4. 记录维修费用和维修时间
5. 完成维修后更新资产状态

### 3.6 盘点管理

#### 3.6.1 盘点计划创建

1. 点击左侧菜单「盘点管理」→「盘点计划」
2. 点击「创建盘点」按钮
3. 设置盘点名称、范围（全公司/部门/位置）
4. 选择盘点时间和负责人
5. 点击「生成盘点清单」

#### 3.6.2 扫码盘点

1. 点击左侧菜单「盘点管理」→「进行盘点」
2. 选择正在进行的盘点任务
3. 使用设备扫描资产二维码
4. 系统自动比对资产信息和预期位置
5. 如发现异常，系统会提示并记录
6. 继续扫描直到完成所有资产盘点

#### 3.6.3 盘点结果处理

1. 完成盘点后，点击「结束盘点」按钮
2. 系统自动生成盘点报告
3. 查看异常资产列表（位置不符、缺失）
4. 根据实际情况处理异常：调整位置、标记丢失等
5. 导出盘点报告存档

### 3.7 报表管理

1. 点击左侧菜单「报表管理」
2. 选择报表类型：
   - 资产统计报表
   - 部门资产分布
   - 资产状态分析
   - 盘点异常报告
3. 设置查询条件和时间范围
4. 点击「生成报表」
5. 可选择导出为Excel或PDF格式

## 4. 移动端使用指南

### 4.1 移动端功能

- 资产扫码查询
- 资产领用登记
- 资产归还登记
- 盘点扫描
- 简单报表查看

### 4.2 使用方法

1. 在移动设备上打开系统网址
2. 登录系统账户
3. 使用内置相机功能扫描资产二维码
4. 按照提示完成相应操作

## 5. 常见问题与故障排除

### 5.1 二维码扫描失败

- 检查光线是否充足
- 确保二维码清晰无损坏
- 调整手机与二维码的距离
- 尝试刷新页面后重试

### 5.2 资产信息无法修改

- 确认用户是否有修改权限
- 检查资产当前状态是否允许修改
- 尝试刷新页面后重试

### 5.3 盘点异常处理

- 位置不符：确认实际位置并更新
- 资产缺失：标记为丢失并通知管理员
- 未登记资产：及时录入系统

## 6. 系统维护与注意事项

### 6.1 数据备份

- 系统会定期自动备份数据
- 管理员可手动进行数据导出备份

### 6.2 安全注意事项

- 定期修改密码
- 不要与他人共享账户
- 离开电脑时请登出系统
- 发现异常请立即联系管理员

### 6.3 最佳实践

- 新购入资产应及时录入系统并生成二维码
- 资产位置变更时及时更新系统信息
- 定期进行资产盘点，建议每季度至少一次
- 妥善保管二维码标签，避免损坏或脱落
