# 二维码标签模板规范

## 1. 标签设计原则

二维码标签是资产身份识别的关键载体，设计应遵循以下原则：

- **唯一性**：每个标签对应唯一的资产编码
- **易读性**：包含必要资产信息，清晰可辨
- **耐久性**：使用耐用材料，适应各种环境
- **统一性**：全公司范围内保持一致的设计规范

## 2. 标签尺寸与材质

### 2.1 推荐尺寸

- **标准尺寸**：3cm × 5cm（适用于大多数中小型设备）
- **大型设备**：5cm × 8cm（适用于大型设备或需要远距离识别的情况）
- **小型设备**：2cm × 3cm（适用于小型设备）

### 2.2 材质选择

- **标准环境**：防水PET不干胶标签（推荐）
- **恶劣环境**：防水防油、耐高温材质
- **金属表面**：专用金属标签或蚀刻方式
- **长期使用**：考虑使用二维码与RFID复合标签

## 3. 标签内容规范

### 3.1 必要信息

1. **公司标识**：公司Logo（左上角）
2. **二维码区域**：占据标签右侧1/2区域
3. **资产编码**：以"AST-"开头的唯一编码
4. **资产名称**：简洁明了的资产名称
5. **资产类型**：设备分类标识

### 3.2 可选信息

1. **品牌型号**：资产的品牌和型号
2. **采购日期**：资产采购时间（可选）
3. **部门标识**：所属部门代码（可选）
4. **标签颜色**：可根据资产类型使用不同颜色区分

## 4. 二维码生成规则

### 4.1 编码格式

采用UUID + Base64编码方式生成唯一标识，具体规则如下：

1. 生成UUID v4格式的唯一标识
2. 对UUID进行Base64编码
3. 结合资产编码生成最终二维码内容
4. 二维码内容格式：`ast:{assetCode}:{encodedUUID}`

### 4.2 二维码技术参数

- **版本**：Version 3
- **纠错级别**：H级（最高纠错率，约30%）
- **模块大小**：根据标签尺寸调整，确保清晰可扫
- **边距**：至少2mm空白边距

## 6. 标签打印规范

### 6.1 打印要求

- **分辨率**：至少300dpi
- **墨水类型**：防水墨水
- **打印方式**：专业标签打印机（推荐）
- **批量打印**：支持模板化批量打印功能

### 6.2 打印流程

1. 从系统导出待打印资产列表
2. 使用专用标签设计软件打开模板
3. 导入资产数据，自动生成二维码
4. 调整打印参数，进行试打印测试
5. 确认无误后批量打印

## 7. 标签粘贴规范

### 7.1 粘贴位置

- **电脑**：主机背面或底部不易摩擦的位置
- **显示器**：背面右下角
- **打印机**：机身侧面或背面
- **办公家具**：不易磨损且容易扫描的位置
- **其他设备**：选择平坦、干燥、不易被覆盖的位置

### 7.2 粘贴方法

1. 确保粘贴表面清洁、干燥、无灰尘
2. 从标签底部开始粘贴，避免气泡
3. 用卡片或手指按压标签表面，确保粘贴牢固
4. 粘贴后24小时内避免沾水或用力摩擦

## 8. 标签维护与更换

### 8.1 定期检查

- 部门管理员应定期检查标签状态
- 发现标签脱落、损坏或模糊不清时应及时更换
- 更换标签时必须保持资产编码不变，重新生成相同内容的二维码

### 8.2 标签更换流程

1. 在系统中查询资产详细信息
2. 确认原始资产编码
3. 生成与原编码相同的新二维码
4. 打印新标签并粘贴
5. 在系统中记录标签更换信息

## 9. 移动端识别优化

为确保移动端设备能够高效识别二维码，应注意以下几点：

- 二维码最小模块尺寸不小于0.1mm
- 标签背景与前景色对比鲜明（推荐黑色二维码，白色背景）
- 避免在曲面或反光表面粘贴
- 对于金属表面，使用专用金属标签或增加底色
