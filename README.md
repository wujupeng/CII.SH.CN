# 企业设备领用与位置追溯系统

## 项目概述

这是一个完整的企业设备领用与位置追溯系统，用于管理企业资产的全生命周期，包括资产录入、领用、归还、维修、盘点和报废等环节。系统提供二维码生成与扫描功能，支持实时定位和追踪资产位置，帮助企业提高资产管理效率，减少资产流失。

## 技术栈

### 前端
- **框架**：React 18
- **路由**：React Router 6
- **状态管理**：Context API + Hooks
- **UI组件**：自定义组件库
- **构建工具**：Vite
- **样式**：原生CSS

### 后端（待实现）
- 基于Node.js/Express或Java Spring Boot
- 数据库：MySQL

## 系统特点

1. **全面的资产管理**：覆盖资产全生命周期
2. **实时定位追踪**：通过二维码实现资产位置精确管理
3. **多级权限控制**：基于角色的访问控制系统
4. **数据可视化**：直观展示各类统计数据和趋势
5. **批量操作**：支持批量导入、导出和处理
6. **响应式设计**：适配多种设备屏幕尺寸
7. **离线工作**：支持离线盘点和数据同步
8. **审计追踪**：完整的操作日志记录

## 安装与运行

### 前端安装

1. 进入frontend目录
```bash
cd frontend
```

2. 安装依赖
```bash
npm install
```

3. 开发环境运行
```bash
npm run dev
```

4. 构建生产版本

## 使用说明

1. **登录系统**：使用分配的用户名和密码登录
2. **初始化设置**：首次使用需要配置部门、用户和资产类型等基础数据
3. **资产录入**：通过手动录入或批量导入添加资产信息
4. **生成二维码**：为新增资产生成唯一二维码标签并贴附
5. **日常管理**：进行领用、归还、维修等日常操作
6. **定期盘点**：创建盘点任务并执行实物盘点
7. **数据分析**：通过报表功能查看各类统计数据

## 注意事项

1. 二维码标签应妥善保管，避免损坏或丢失
2. 盘点操作应定期执行，确保资产数据准确性
3. 管理员权限应谨慎分配，避免权限滥用
4. 重要数据应定期备份
5. 系统使用过程中遇到问题请及时联系技术支持
